# Package index

## All functions

- [`attributable()`](https://pasahe.github.io/bdlnm/reference/attributable.md)
  : Calculate attributable number and fractions from a Bayesian
  distributed-lag model (B-DLNM).
- [`bcrosspred()`](https://pasahe.github.io/bdlnm/reference/bcrosspred.md)
  : Predict exposure-lag-response effects from a Bayesian
  distributed-lag model (B-DLNM).
- [`bdlnm()`](https://pasahe.github.io/bdlnm/reference/bdlnm.md) : Fit a
  Bayesian distributed lag non-linear model (B-DLNM)
- [`london`](https://pasahe.github.io/bdlnm/reference/london.md) :
  London temperature and mortality data
- [`optimal_exposure()`](https://pasahe.github.io/bdlnm/reference/optimal_exposure.md)
  : Calculate the exposure value that minimizes or maximizes the overall
  cumulative effect of a Bayesian distributed lag non-linear model
  (B-DLNM)
- [`plot(`*`<bcrosspred>`*`)`](https://pasahe.github.io/bdlnm/reference/plot.bcrosspred.md)
  : Plot the predicted effects from a Bayesian distributed-lag model
  (B-DLNM).
- [`plot(`*`<optimal_exposure>`*`)`](https://pasahe.github.io/bdlnm/reference/plot.optimal_exposure.md)
  : Plot posterior distribution of optimal effect exposure values
